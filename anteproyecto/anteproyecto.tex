\begin{frame}{Metodología propuesta}

\vspace{0.25cm}

\begin{columns}[T]

\column{0.48\textwidth}

\textbf{Análisis comparativo}

\begin{itemize}
    \item Prueba t para comparar promedio entre bajo uso y alto uso de redes.
    \item ANOVA para comparar promedio entre varias categorías de uso.
\end{itemize}

\vspace{0.2cm}


\column{0.48\textwidth}

\textbf{Análisis de asociación}

\begin{itemize}
    \item Correlación de Pearson o Spearman entre horas en redes y promedio.
\end{itemize}




\end{columns}

\vspace{0.25cm}

\begin{alertblock}{Alternativa descartada}
Consideramos clasificar por red social principal, especialmente YouTube por su posible uso académico,
pero lo descartamos como eje central porque la pregunta principal se responde mejor desde la intensidad de uso,
no desde la plataforma.
\end{alertblock}

\end{frame}

%-----------------------------------------------------------------------------------------

\begin{frame}{Resultados esperados}

\small

\begin{columns}[T]

\column{0.52\textwidth}

\textbf{Patrones esperados}

\begin{itemize}
    \item Diferencias en rendimiento académico entre grupos con distinto uso de redes sociales.
    \item Mayor dispersión y promedios ligeramente menores en estudiantes con uso elevado.
    \item Asociación positiva entre horas de estudio y rendimiento académico.
\end{itemize}

\column{0.42\textwidth}

\begin{block}{Idea central}
No esperamos necesariamente una relación lineal fuerte, sino una asociación condicionada
por la distracción académica y las horas de estudio.
\end{block}

\end{columns}

\vspace{0.3cm}

\begin{alertblock}{Criterio para responder la pregunta}
Consideraremos respondida la pregunta si las pruebas estadísticas y el modelo permiten determinar
si el uso de redes sociales, solo o combinado con distracción académica, se asocia significativamente
con el promedio académico.
\end{alertblock}
